🧊 Glassmorphism Card
<div class="bg-white/10 backdrop-blur-lg border border-white/30 rounded-xl"></div>

🧠 Interview Q&A

Q1: Difference between filter and backdrop-filter?
→ filter affects the element itself, while backdrop-filter affects what’s behind it.

Q2: Can multiple filters be combined?
→ Yes! Example: filter: blur(5px) brightness(1.2);

Q3: How to create glassmorphism in Tailwind?
→ Combine bg-white/10, backdrop-blur, border-white/30, and rounded-xl.
