üîµ Circle

circle

Centered circle with radius: 50%.

/* Tailwind */
<div class="w-48 h-48 bg-orange-400 [clip-path:circle(50%_at_50%_50%)]"></div>

/* Plain CSS */
.element { clip-path: circle(50% at 50% 50%); }

üü£ Ellipse

ellipse

Horizontal radius 50%, vertical radius 30%.

<div class="w-72 h-48 bg-violet-400 [clip-path:ellipse(50%_30%_at_50%_50%)]"></div>

/* CSS */
.element { clip-path: ellipse(50% 30% at 50% 50%); }

üî∑ Polygon (Triangle / Diamond)

triangle
<div class="w-40 h-40 bg-sky-400 [clip-path:polygon(50%_0%,0%_100%,100%_100%)]"></div>
diamond
<div class="w-40 h-40 bg-indigo-400 [clip-path:polygon(50%_0%,100%_50%,50%_100%,0%_50%)]"></div>

Tip: polygon points are connected in order ‚Äî try editing the coordinates to craft custom shapes.

üü• Inset

inset

Cuts the box inward by top/right/bottom/left values.

<div class="w-72 h-48 bg-red-400 [clip-path:inset(10px_20px_30px_40px)]"></div>

/* CSS */
.element { clip-path: inset(10px 20px 30px 40px); }

üü° Path (SVG)

path

Use SVG path data. When using Tailwind arbitrary values, replace spaces with underscores (_).

<!-- Tailwind (spaces replaced with underscores) -->
<div class="w-48 h-48 bg-yellow-400 [clip-path:path('M10_10_h80_v80_h-80_Z')]"></div>

/* Plain CSS */
.element { clip-path: path('M10 10 h80 v80 h-80 Z'); }

Note: Browser support for clip-path: path() is improving but may vary. Use fallbacks if needed.
